# Precios Crypto (CoinGecko)

Consulta precios actuales de cualquier criptomoneda en USD, MXN u otras divisas. Sin API key requerida.

## Tools
- `crypto_price` — GET `https://api.coingecko.com/api/v3/simple/price`

## Settings
Ninguno.

## Capabilities
- `http:fetch api.coingecko.com`

## Install
```bash
/tmp/metacore_cli validate .
/tmp/metacore_cli build .
```

Runtime: `webhook` · Kernel: `>=2.0.0 <3.0.0` · Tenant isolation: `shared`
